# Qayrawan Beta

This repository contains the working static beta for the **Qayrawan** school platform.

## What it includes
- Login system
- 1 teacher/admin account
- 5 student accounts
- Announcements
- Homework grouped by grade and class
- Discussions with replies
- Profile system with preset avatars only
- Settings, maintenance mode, audit log, and backup export/import

## Important note
This beta is designed for **GitHub Pages** and uses browser storage.  
Backup/import keeps data portable between devices.
